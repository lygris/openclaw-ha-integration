# OpenClaw Home Assistant Integration

Custom Home Assistant integration for connecting Assist to an OpenClaw agent with stable session routing.

## Features
- Native Home Assistant conversation integration
- Configurable OpenClaw base URL, token, agent id, and stable session id
- Compact entity-context enrichment for explicitly matched entities
- Conservative structured action contract for optional Home Assistant service execution

## Installation
Install through HACS as a custom repository or copy `custom_components/openclaw` into your Home Assistant config.
